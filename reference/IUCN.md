# IUCN evaluation for frogs of the genus Phyllomedusa

Result of the function lets.iucn (deprecated) applied to South American
frog genus Phyllomedusa in 2014.

## Usage

``` r
IUCN
```

## Format

A data frame with 32 rows and 7 columns:

- Species:

  Scientific name

- Family:

  Family

- Status:

  Red List Status

- Criteria:

  Criteria for listing as threatened

- Population:

  Population trend per IUCN

- Description_Year:

  Year described

- Country:

  Presence in country

## Source

IUCN - <https://www.iucnredlist.org/>. 2014.
