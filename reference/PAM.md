# PresenceAbsence object for frogs of the genus Phyllomedusa

PresenceAbsence object obtained using the function
[`lets.presab`](https://brunovilela.github.io/letsR/reference/lets.presab.md)
for the Geographic distribution of the South American frog genus
[Phyllomedusa](https://brunovilela.github.io/letsR/reference/Phyllomedusa.md).

## Usage

``` r
PAM
```

## Format

A PresenceAbsence object

## Source

Generated from IUCN Spatial Data - <https://www.iucnredlist.org/>. 2014.
